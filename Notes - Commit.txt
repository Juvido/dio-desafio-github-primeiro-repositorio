Commit - Adicionar arquivos no repositório:

 - Comandos
git add . ou git commit -a => todos os arquivos, não inclusos, serão adicionados
			ao controle de versão
git commit -m "msg a ser escrita"  =>os arquivos serão commitados 
			para a nuvem - ainda não está atualizado no 
			repositório
git push origin main => enviará os commits para a nuvem e atualizará
			o repositório no GitHub

git status => verifica as atualizações do diretório criado e se há 
			alguma atualização pendente
